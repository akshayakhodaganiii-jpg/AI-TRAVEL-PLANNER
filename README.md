# AI Travel Planner ✈️

An AI-powered travel planning application that helps users generate personalized itineraries, discover destinations, and plan trips based on their preferences and budget.

## 🌟 Features

- 🧭 **Smart Itinerary Generation** – Get a day-by-day travel plan tailored to your destination, budget, and trip duration
- 📍 **Destination Recommendations** – Discover attractions, restaurants, and hidden gems
- 💰 **Budget-Aware Planning** – Suggestions adapted to your spending range
- 🗓️ **Flexible Trip Lengths** – Plans for weekend getaways or multi-week adventures
- 🎨 **Simple, Intuitive Interface** – Enter a prompt or preferences and get a ready-to-use plan

## 📦 Installation

1. Clone the repository
   ```bash
   git clone https://github.com/akshayakhodaganiii-jpg/AI-TRAVEL-PLANNER.git
   cd AI-TRAVEL-PLANNER
   ```

2. Install dependencies
   ```bash
   npm install
   # or
   pip install -r requirements.txt
   ```

3. Set up environment variables

   Create a `.env` file in the root directory and add your API keys:
   ```
   AI_API_KEY=your_api_key_here
   MAPS_API_KEY=your_maps_api_key_here
   ```

4. Run the application
   ```bash
   npm start
   # or
   streamlit run app.py
   ```

## 🚀 Usage

1. Launch the app
2. Enter your trip details (destination, dates, budget, preferences)
3. Let the AI generate a personalized itinerary
4. Review, customize, and save your travel plan
